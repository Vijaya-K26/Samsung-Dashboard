📱 Samsung Sales Dashboard

A modern, interactive Power BI dashboard designed to analyze Samsung's sales performance, customer behavior, inventory, shipments, production, and procurement. The dashboard provides business insights through KPIs, interactive charts, drillthrough navigation, and dynamic filtering.

📌 Project Overview

This project demonstrates end-to-end Business Intelligence using Power BI with a star schema data model. It helps stakeholders monitor sales performance, identify top-performing products, analyze customer trends, and track inventory and supply chain operations.
